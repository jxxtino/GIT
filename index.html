Estudos sobre GIT

Git será usado para o versionamento de nosso código, através dele podemos criar versões da aplicação, podendo assim previnir erros e proporcionar testes sem comprometer o funcionamento geral.

Comandos GIT

Iniciar GIT: git init Agrupar mudanças: git add . Criar um Commit: git commit -m {nome do commit} Verificar commits: git log Status do código: git status Comparar diferenças: git diff Restaurar todo o commit: git restore . Restaurar o commit especifico: git restore name.name Restaurar o staged: git restore --staged Restaurar o staged especifico: git restore --staged name.name mudar título do commit: git commit --amend "name" excluir o último commit: git reset --soft HEAD~1
